Newspaper App
This newspaper website uses django for backend framework and Bootstrap4 and CSS for frontend. One can login to their account and create and edit their own articles. It uses django and crispy forms to modify the login and signup pages.
Project Details / Background
This is a newspaper website powered with Django, Bootstrap5, CSS. It utilizes several inbuilt Django forms like ArticleListView, ArticleUpdateView and so on to model a convenient interface for users.
It also utilizes Django's model feature to store database for the articles and comments. I have used django's many-to-one relationship between articles and comments to relate their databases. For the frontend, the website relies on Bootstrap and a hint of CSS styling. To connect the backend and frontend, it utilizes Django's algorithm for URL systems.
