Real Time Object Detection System
It is an ultralytics YOLO (You Only Look Once) based real time object detection model which utilizes openCV and object detection loop and has the potential of identifying 80 different objects.
Project Details / Background
I loaded the yolov8m.pt model into the code which utilizes its deep learning technique to divide an image into grids and predicts the bounding boxes and class probabilities of objects in the grid with great accuracy. I used OpenCV to read video frames, display frames with detected objects and help in rendering bounding boxes and labels.
The object detection loop continuously reads frames from the video while YOLO and openCV are busy extracting the data for boxes and class. After YOLO and openCV perform object detection, I have used numpy to convert the extracted bounding boxes and class information into numpy arrays for efficient data manipulation.
Demos

I tested out different models of YOLOv8 on its accuracy and speed. First, I tried to test it on a self-recorded video of my university with fewer objects in the video.

Then I tried to test it on a self-recorded video with comparatively more objects and placed at a farther distance. I have used the yolov8m model on the left side and yolov8x model on the right side. It seemed like the yolov8m.pt model couldn't recognize objects at a farther distance, so I decided to use its yolov8x.pt model. With this model, I could detect objects concisely but it demanded additional GPU and time to execute. I have zoomed into the areas of the video where the detection efficiency is more apparent.
